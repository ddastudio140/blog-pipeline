# 목적

- 블로그 글 작성에 필요한 키워드를 선정하고
- 네이버 뉴스에서 키워드로 데이터를 수집한다.
- 수집된 데이터를 기반으로 블로그 글을 마크다운 형식으로 작성한다.
- 작성한 블로그는 github에 푸시한다.

# n8n

- 기존에는 n8n으로 위 기능들을 구현했는데
- n8n에서 파이썬으로 대체하려고 함.
- auto-blog.json 파일을 참고할 것

# 특이사항

- 블로그 글을 쓸 때는 https://build.nvidia.com/ 여기서 발급 받은 api key 를 사용해서 수행한다.
- 기존 n8n 에서는 특정 사이트에서 키워드를 가져오고
- 사용자가 특정 키워드를 전송할 수도 있었다.

# 고려사항

- 사용자가 데이터 수집 주기를 언제 얼마마다 실행지 설정할 수 있어야 한다.
- 작성된 블로그 글의 제목, 요약 정보를 어딘가에 저장할 수 있어야 한다.
  - 새로운 블로그 글을 작성할 때 과거에 작성한 글을 참조 형태를 넣기 위한 목적

# 기억해야 할 것

- 프로젝트 분석 및 설계 작업은 fable을 이용하고
- 실제 구현 계획이 나오면 코드 작업은 sonnet을 이용한다.
